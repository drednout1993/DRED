# Система учёта реестров документов
## Финансово-экономический отдел

Десктопное приложение на Python для автоматизации работы с реестрами документов.

### Требования

- Python 3.11+
- MySQL 8.0+
- Laragon (для Windows)

### Установка

1. Создать виртуальное окружение:
```bash
python -m venv venv
```

2. Активировать виртуальное окружение:
```bash
# Windows
venv\Scripts\activate

# Linux/Mac
source venv/bin/activate
```

3. Установить зависимости:
```bash
pip install -r requirements.txt
```

4. Настроить базу данных (см. database/create_tables.sql)

5. Запустить приложение:
```bash
python main.py
```

### Структура проекта

```
reestr_system/
├── main.py              # Точка входа
├── config.py            # Конфигурация

├── uploads/             # Загрузки
└── generated_pdfs/      # PDF документы
```

### Роли пользователей

- **Администратор** — полный контроль системы
- **Финансист** — проверка и принятие реестров
- **Руководитель ФЭО** — те же права что у финансиста
- **Пользователь** — создание и редактирование своих реестров
